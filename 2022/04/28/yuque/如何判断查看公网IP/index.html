如何判断查看公网IP

https://www.jianshu.com/p/a0aced8652c8

前置条件

  • 你可以电脑本机 PPPOE 拨号。然后 任务管理器（Win10） 查看。
  • 你可以登录光猫查看。（需要账号密码，一般在光猫盒底下）
  • 你可以登录路由器查看。（需要账号密码，一般为自己设置或路由器底下）

如何判断

判断准则：一致就是公网 IP，反之。

这里推荐 ip.sb 来查看
查看方法，浏览器 地址栏 中输入，会同时得到你的 IPV4 和 IPV6 地址

PPPOE 比较少 不展开

看 任务管理器 ，是否与 ip.sb 一致。

光猫

打开 状态-网络侧信息 查看 INTERNET_R_VID 的 IP 地址 是否与 ip.sb 一致。

路由器

利用 路由器 账号密码 登录到路由器，查看 WAN 口状态 中的 IP 地址 是否与 ip.sb 一致。
